# Valax

Valax is a free Discord bot communication workspace by Jotoro. This repository currently contains
the public English landing page and interactive product preview.

## Local preview

Run the zero-dependency Node.js preview server:

```powershell
npm run dev
```

The site is static and can be deployed directly to Vercel. Discord authentication, bot token
handling, and MySQL integration are intentionally not part of this landing-page milestone.
